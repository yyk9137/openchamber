# Заплановані завдання

Заплановане завдання запускає промпт за вас за розкладом — наприклад, щоденне «підсумуй вчорашні зміни» чи щотижневе прибирання. Коли воно спрацьовує, OpenChamber починає нову сесію й надсилає промпт самостійно. Відкрийте планувальник із кнопки вгорі бічної панелі сесій.

## Створення завдання

1. Відкрийте діалог запланованих завдань із бічної панелі сесій.
2. Додайте завдання й дайте йому назву.
3. Виберіть, коли воно запускається:
   - **daily** — в один або кілька моментів щодня
   - **weekly** — у вибрані дні тижня та час
   - **once** — одна дата й час
4. Задайте, що воно робить: промпт для надсилання, а також провайдера, модель і агента для використання. Промпт може бути слеш-командою, як-от `/review`.
5. Збережіть і переконайтеся, що завдання ввімкнене.

Ви можете запустити будь-яке завдання негайно через **run now**, щоб перевірити, що воно робить те, що ви очікуєте.

## Як виглядає успіх

Після запуску завдання показує, коли воно востаннє виконувалося, чи воно успішне, і посилання на сесію, яку воно створило. Якщо запуск збоїть, помилка теж показується там.

## Майте на увазі

Завдання спрацьовують лише поки запущений сервер OpenChamber. Якщо ви його закриєте, заплановані запуски призупиняються, доки він не повернеться.

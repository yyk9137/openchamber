# Голосовий режим

Голосовий режим дає змогу диктувати повідомлення й слухати відповіді, зачитані вам уголос. Увімкніть його в **Settings → OpenChamber → Voice**.

## Зачитування відповідей уголос (текст-у-мовлення)

Виберіть, як озвучуються відповіді:

- **browser** — вбудовані голоси вашого браузера, без налаштування
- **OpenAI** — голоси OpenAI; вставте свій API-ключ і виберіть голос
- **OpenAI-compatible** — будь-який сервіс, що говорить у форматі OpenAI; уведіть його URL і, якщо потрібно, API-ключ
- **macOS say** — вбудована команда `say`, на Mac'ах, де вона є

Коли ввімкнено, ви отримуєте кнопку відтворення на повідомленнях, щоб почути їх зачитаними вголос.

## Диктування повідомлень (мовлення-у-текст)

Виберіть, як транскрибується ваше мовлення:

- **browser** — вбудоване розпізнавання вашого браузера, без налаштування
- **server** — OpenAI-сумісний сервіс транскрипції; уведіть його URL і, за потреби, API-ключ
- **on-device** — модель мовлення, що працює у вашому браузері й завантажується при першому використанні

## Зауваження для телефонів

Для зачитування відповідей уголос на телефоні опції OpenAI чи OpenAI-compatible найнадійніші — мобільні браузери обмежують вбудовані голоси.

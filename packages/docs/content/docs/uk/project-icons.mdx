# Іконки проєктів

Іконка проєкту допомагає легко відрізняти ваші проєкти з першого погляду. OpenChamber намагається знайти її за вас, а ви завжди можете задати власну. Керуйте нею в **Settings → Projects**.

## Автоматичне виявлення

Коли ви додаєте проєкт, OpenChamber шукає файл `favicon` усередині нього й використовує його як іконку проєкту. Якщо ваш репозиторій уже постачається з favicon, іконка зазвичай просто з'являється — нічого робити не треба.

## Задати власну

Відкрийте **Settings → Projects** і завантажте зображення (PNG, JPEG чи SVG, до 5 МБ). Власне зображення має пріоритет над автоматично виявленим. Ви також можете натомість вибрати колір або прибрати зображення, щоб повернутися до виявлення.

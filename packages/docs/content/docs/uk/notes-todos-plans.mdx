# Нотатки, завдання та плани проєкту

Кожен проєкт має власний робочий простір для нотаток, списку завдань і збережених планів. Вони належать проєкту, а не якійсь окремій сесії, тож зберігаються, коли ви переходите між сесіями. Знайдіть їх на вкладці **Context** у правій бічній панелі (окрема вкладка на мобільному).

## Нотатки

Поле для довільних нотаток про все, що ви хочете запам'ятати про проєкт. Воно зберігається саме в міру набирання.

## Завдання

Простий чеклист. Додавайте пункти, відмічайте виконані, переставляйте їх перетягуванням і очищайте завершені.

Кожне завдання має меню **send**, щоб ви могли передати його агентові:

- надіслати в поточну сесію
- почати з ним нову сесію
- почати з ним нову [сесію worktree](/uk/worktrees/) (лише коли проєкт є git-репозиторієм)

## Плани

Місце, щоб тримати довші плани як збережені файли. Ви можете:

- імпортувати план із файлу Markdown чи тексту
- відкрити план, щоб прочитати його в бічній панелі
- видаляти плани, які вам більше не потрібні

Ви маєте повернутися на вкладку Context зі збереженою нотаткою, відміченим завданням або переліченим планом — так ви розумієте, що все спрацювало.

## Пов'язане

- [Сесії worktree](/uk/worktrees/) — запускайте завдання у власній гілці
- [Проєкти](/uk/projects/) — це належить активному проєкту

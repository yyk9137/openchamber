# Використання та квоти

Сторінка «Використання» показує, скільки з плану кожного провайдера ви витратили, щоб ви бачили, наскільки близько до ліміту перебуваєте. Відкрийте її в **Settings → Usage**.

## Що ви бачите

Для кожного підключеного провайдера OpenChamber показує:

- скільки з поточного вікна ви використали, у вигляді смужки
- розподіл за моделями
- індикатор темпу, щоб ви могли зрозуміти, чи встигнете вичерпати ліміт

Ви можете обрати, які провайдери відображаються, і той самий підсумок доступний із випадного меню в заголовку застосунку.

## Підтримувані провайдери

Використання працює для провайдерів, які публікують квоту, зокрема Claude, Codex, GitHub Copilot, Google, OpenRouter, Kimi, NanoGPT, z.ai, Zhipu, MiniMax, Ollama Cloud та Wafer.

Провайдер показує використання лише після того, як ви до нього увійшли на сторінці [Провайдери](/uk/providers/). Кільком провайдерам потрібен додатковий крок — наприклад, Ollama Cloud читає файл сесії, який ви налаштовуєте окремо. Якщо провайдер не показує даних, зазвичай це означає, що бракує цих додаткових облікових даних.

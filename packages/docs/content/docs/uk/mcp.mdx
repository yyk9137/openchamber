# MCP-сервери

MCP-сервер дає вашим агентам додаткові інструменти — як-от пошук у базі даних, виклик API чи читання сервісу, яким ви користуєтеся. Додавайте їх у **Settings → MCP**.

## Додавання сервера

1. Відкрийте **Settings → MCP**.
2. Додайте сервер і виберіть його тип:
   - **local** — OpenChamber запускає команду на вашій машині. Ви вказуєте йому команду для запуску і, за потреби, змінні середовища.
   - **remote** — OpenChamber підключається до URL, який хостить хтось інший. Ви вказуєте йому URL і будь-які потрібні заголовки (наприклад, токен автентифікації).
3. Збережіть. Сервер увімкнений за замовчуванням; ви можете вимкнути його, не видаляючи.

## Де він застосовується

Виберіть область при додаванні сервера:

- **personal** — доступний у кожному проєкті
- **project** — доступний лише в поточному проєкті й зберігається разом з іншими налаштуваннями проєкту

Імена серверів використовують малі літери, цифри, дефіси та підкреслення.

# Навички

Навичка — це багаторазовий набір інструкцій, який агент може підтягнути, коли він доречний, — наприклад, «як ми пишемо повідомлення комітів» або «наші конвенції API». Керуйте ними в **Settings → Skills**.

Щоб установити готові навички замість писати власні, див. [Каталог навичок](/uk/skills-catalog/).

## Створення навички

1. Відкрийте **Settings → Skills**.
2. Створіть навичку й дайте їй назву та короткий опис. За описом агент вирішує, коли навичка застосовна, тож робіть його конкретним.
3. Напишіть інструкції. Додайте допоміжні файли, якщо навичка їх потребує.
4. Виберіть, де вона зберігається:
   - **personal** — доступна в кожному проєкті
   - **project** — доступна лише в поточному проєкті

## Використання навички в чаті

Введіть `/` посередині повідомлення, щоб викликати перемикач навичок, потім оберіть одну. Агент завантажить інструкції цієї навички для відповіді.

`/` на самому початку повідомлення натомість відкриває [команди](/uk/commands-snippets/).

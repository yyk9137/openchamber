# 예약 작업

예약 작업은 일정에 따라 프롬프트를 대신 실행합니다. 예를 들어 매일 "어제 변경 사항 요약"이나 주간 정리 같은 것입니다. 실행되면 OpenChamber가 새 세션을 시작하고 프롬프트를 알아서 보냅니다. 세션 사이드바 상단의 버튼에서 스케줄러를 엽니다.

## 작업 만들기

1. 세션 사이드바에서 예약 작업 대화 상자를 엽니다.
2. 작업을 추가하고 이름을 지정합니다.
3. 실행 시점을 선택합니다.
   - **daily** — 매일 하나 이상의 시각에
   - **weekly** — 선택한 요일과 시각에
   - **once** — 단일 날짜와 시각에
4. 동작을 설정합니다. 보낼 프롬프트와 사용할 공급자, 모델, 에이전트를 지정합니다. 프롬프트는 `/review`처럼 슬래시 명령일 수 있습니다.
5. 저장하고 작업이 활성화되어 있는지 확인합니다.

**run now**로 작업을 즉시 실행하여 기대대로 동작하는지 확인할 수 있습니다.

## 성공이란 어떤 모습인가

실행 후 작업에는 마지막 실행 시각, 성공 여부, 생성된 세션으로의 링크가 표시됩니다. 실행이 실패하면 오류도 거기에 표시됩니다.

## 유의할 점

작업은 OpenChamber 서버가 실행 중일 때만 동작합니다. 서버를 닫으면 다시 켤 때까지 예약 실행이 일시 중지됩니다.

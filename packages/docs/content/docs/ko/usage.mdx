# 사용량 및 할당량

Usage 페이지는 각 공급자 요금제를 얼마나 사용했는지 보여주므로 한도에 얼마나 가까운지 확인할 수 있습니다. **Settings → Usage**에서 엽니다.

## 표시되는 내용

연결된 각 공급자에 대해 OpenChamber는 다음을 표시합니다.

- 현재 기간 동안 사용한 양을 막대로 표시
- 모델별 분석
- 한도에 도달할 추세인지 알 수 있는 사용 속도 지표

어떤 공급자를 표시할지 선택할 수 있으며, 동일한 요약은 앱 헤더의 드롭다운에서도 확인할 수 있습니다.

## 지원되는 공급자

Usage는 Claude, Codex, GitHub Copilot, Google, OpenRouter, Kimi, NanoGPT, z.ai, Zhipu, MiniMax, Ollama Cloud, Wafer를 포함해 할당량을 공개하는 공급자에서 작동합니다.

공급자는 [Providers](/ko/providers/) 페이지에서 로그인한 뒤에만 사용량을 표시합니다. 일부 공급자는 추가 단계가 필요합니다. 예를 들어 Ollama Cloud는 별도로 설정한 세션 파일을 읽습니다. 공급자에 데이터가 표시되지 않는다면 대개 그 추가 자격 증명이 누락된 것입니다.

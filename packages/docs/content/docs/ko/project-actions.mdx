# 프로젝트 작업

프로젝트 액션은 한 번 저장해 두고 클릭으로 실행하는 셸 명령입니다. 개발 서버, 빌드, 테스트 실행 등이 해당됩니다. 각 프로젝트는 자체 목록을 유지합니다. **Settings → Projects → Project Actions**에서 설정합니다.

## 액션 추가하기

1. **Settings → Projects**를 열고 **Project Actions** 섹션을 찾습니다.
2. 액션을 추가하고 이름을 지정하고 아이콘을 선택하고 실행할 명령을 입력합니다.
3. 저장합니다.

명령이 특정 운영체제에서만 의미가 있다면 액션을 해당 운영체제로 제한할 수 있습니다.

## 액션 실행하기

액션은 앱 헤더의 메뉴에 있습니다. 하나를 클릭하면 OpenChamber가 프로젝트 폴더의 터미널에서 실행하고, 출력을 볼 수 있도록 터미널 보기로 전환합니다. 같은 메뉴에서 중지할 수 있습니다.

## 개발 서버 자동 열기

서버를 시작하는 액션에 **auto-open URL**을 켭니다. OpenChamber가 출력에서 로컬 주소를 감시하다가 열기를 제안합니다. [Preview & Dev Servers](/ko/preview/)를 참고하세요. 데스크톱에서는 SSH 포트 포워딩을 통해 라우팅할 수도 있습니다.

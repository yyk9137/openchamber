# Powiadomienia

Powiadomienia informują Cię, gdy coś wymaga Twojej uwagi, dzięki czemu nie musisz wpatrywać się w ekran — sesja się zakończyła, napotkała błąd, zadała pytanie albo potrzebuje uprawnień do zrobienia czegoś. Skonfiguruj je w **Settings → OpenChamber → Notifications**.

## Włącz je

1. Otwórz **Settings → OpenChamber → Notifications**.
2. Zezwól na powiadomienia, gdy przeglądarka lub system o to poprosi.
3. Wybierz, o czym chcesz być informowany:
   - sesja się **kończy**
   - sesja napotyka **błąd**
   - sesja **zadaje pytanie**
   - sesja potrzebuje **uprawnień**
   - **podzadania** się kończą

## Jak do Ciebie docierają

- na **komputerze** otrzymujesz natywne powiadomienia systemowe
- w **przeglądarce lub zainstalowanej aplikacji** otrzymujesz powiadomienia web push, więc docierają nawet wtedy, gdy karta jest w tle

Sesje ustawione na automatyczną akceptację nie będą Cię nagabywać powiadomieniami o uprawnieniach.

## Dostosuj treść

Każdy rodzaj powiadomienia ma szablon tytułu i wiadomości, który możesz edytować, używając pól takich jak nazwa agenta i model. Jest też limit tego, ile z ostatniej wiadomości zostaje dołączone, aby powiadomienia pozostały krótkie.

# Ikony projektów

Ikona projektu ułatwia odróżnienie projektów na pierwszy rzut oka. OpenChamber próbuje znaleźć jedną za Ciebie, a Ty zawsze możesz ustawić własną. Zarządzaj nią w **Settings → Projects**.

## Automatyczne wykrywanie

Gdy dodajesz projekt, OpenChamber szuka w nim pliku `favicon` i używa go jako ikony projektu. Jeśli Twoje repozytorium już zawiera favicon, ikona zwykle po prostu się pojawia — nic nie trzeba robić.

## Ustaw własną

Otwórz **Settings → Projects** i prześlij obraz (PNG, JPEG lub SVG, do 5 MB). Własny obraz ma pierwszeństwo przed wykrytym automatycznie. Możesz też wybrać kolor albo usunąć obraz, aby wrócić do wykrywania.

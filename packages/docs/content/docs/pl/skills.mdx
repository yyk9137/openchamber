# Skille

Skill to wielokrotnego użytku zestaw instrukcji, który agent może wczytać, gdy jest istotny — na przykład „jak piszemy komunikaty commitów” albo „nasze konwencje API”. Zarządzasz nimi w **Settings → Skills**.

Aby zainstalować gotowe skille zamiast pisać własne, zobacz [Katalog skilli](/pl/skills-catalog/).

## Utwórz skill

1. Otwórz **Settings → Skills**.
2. Utwórz skill i nadaj mu nazwę oraz krótki opis. Na podstawie opisu agent decyduje, kiedy skill ma zastosowanie, więc niech będzie konkretny.
3. Napisz instrukcje. Dodaj pliki pomocnicze, jeśli skill ich potrzebuje.
4. Wybierz, gdzie ma się znajdować:
   - **personal** — dostępny w każdym projekcie
   - **project** — dostępny tylko w bieżącym projekcie

## Użyj skilla w czacie

Wpisz `/` w środku wiadomości, aby otworzyć selektor skilli, a następnie wybierz jeden. Agent wczyta instrukcje tego skilla na potrzeby odpowiedzi.

`/` na samym początku wiadomości otwiera natomiast [polecenia](/pl/commands-snippets/).

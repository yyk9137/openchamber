# Zużycie i limity

Strona zużycia pokazuje, ile planu każdego dostawcy wykorzystałeś, dzięki czemu widzisz, jak blisko limitu jesteś. Otwórz ją w **Settings → Usage**.

## Co widzisz

Dla każdego połączonego dostawcy OpenChamber pokazuje:

- ile bieżącego okna wykorzystałeś, w postaci paska
- podział według modelu
- wskaźnik tempa, byś mógł ocenić, czy zmierzasz w stronę wyczerpania limitu

Możesz wybrać, którzy dostawcy się pojawiają, a to samo podsumowanie jest dostępne z rozwijanego menu w nagłówku aplikacji.

## Obsługiwani dostawcy

Zużycie działa dla dostawców, którzy publikują limit, w tym Claude, Codex, GitHub Copilot, Google, OpenRouter, Kimi, NanoGPT, z.ai, Zhipu, MiniMax, Ollama Cloud i Wafer.

Dostawca pokazuje zużycie dopiero po zalogowaniu się do niego na stronie [Dostawcy](/pl/providers/). Kilku dostawców wymaga dodatkowego kroku — na przykład Ollama Cloud odczytuje plik sesji skonfigurowany osobno. Jeśli dostawca nie pokazuje danych, zwykle oznacza to brak tego dodatkowego poświadczenia.

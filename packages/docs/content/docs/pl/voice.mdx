# Tryb głosowy

Tryb głosowy pozwala dyktować wiadomości i odsłuchiwać odpowiedzi. Włącz go w **Settings → OpenChamber → Voice**.

## Odczytywanie odpowiedzi na głos (text-to-speech)

Wybierz, jak odpowiedzi są wymawiane:

- **browser** — wbudowane głosy Twojej przeglądarki, bez konfiguracji
- **OpenAI** — głosy OpenAI; wklej swój klucz API i wybierz głos
- **OpenAI-compatible** — dowolna usługa mówiąca w formacie OpenAI; wprowadź jej adres URL oraz, jeśli go potrzebuje, klucz API
- **macOS say** — wbudowane polecenie `say`, na Macach, które je mają

Po włączeniu na wiadomościach pojawia się przycisk odtwarzania, aby je odsłuchać.

## Dyktowanie wiadomości (speech-to-text)

Wybierz, jak Twoja mowa jest transkrybowana:

- **browser** — wbudowane rozpoznawanie Twojej przeglądarki, bez konfiguracji
- **server** — usługa transkrypcji zgodna z OpenAI; wprowadź jej adres URL oraz, w razie potrzeby, klucz API
- **on-device** — model mowy, który działa w Twojej przeglądarce i pobiera się przy pierwszym użyciu

## Uwaga dla telefonów

Do odczytywania odpowiedzi na głos na telefonie najbardziej niezawodne są opcje OpenAI lub OpenAI-compatible — przeglądarki mobilne ograniczają wbudowane głosy.

# Catálogo de Skills

O Catálogo de Skills permite instalar skills que outras pessoas publicaram em vez de escrever as suas. Abra-o em **Settings → Skills → Catalog**.

Para escrever suas próprias skills, veja [Skills](/pt-br/skills/).

## Instalar uma skill

1. Abra o catálogo.
2. Navegue pelas fontes integradas — o repositório de skills da Anthropic e o registro comunitário ClawdHub — ou pesquise.
3. Escolha uma skill e instale-a.
4. Escolha onde instalá-la: para tudo o que você faz, ou apenas no projeto atual.

Se já existir uma skill com o mesmo nome, o OpenChamber pergunta o que fazer — pular, sobrescrever ou decidir por skill.

## Adicionar sua própria fonte

Você pode adicionar qualquer repositório Git como fonte usando seu nome `owner/repo` ou uma URL Git completa. Repositórios privados precisam de acesso configurado na sua máquina (uma chave SSH ou credenciais salvas); se uma fonte não conseguir autenticar, o catálogo avisa em vez de falhar silenciosamente.

# Modo de Voz

O modo de voz permite ditar mensagens e ter as respostas lidas de volta para você. Ative-o em **Settings → OpenChamber → Voice**.

## Ler respostas em voz alta (text-to-speech)

Escolha como as respostas são faladas:

- **browser** — as vozes integradas do seu navegador, sem configuração
- **OpenAI** — as vozes da OpenAI; cole sua chave de API e escolha uma voz
- **OpenAI-compatible** — qualquer serviço que fale no formato da OpenAI; insira sua URL e, se precisar, uma chave de API
- **macOS say** — o comando integrado `say`, em Macs que o têm

Uma vez ativado, você ganha um botão de play nas mensagens para ouvi-las lidas em voz alta.

## Ditar mensagens (speech-to-text)

Escolha como sua fala é transcrita:

- **browser** — o reconhecimento integrado do seu navegador, sem configuração
- **server** — um serviço de transcrição compatível com OpenAI; insira sua URL e, se necessário, uma chave de API
- **on-device** — um modelo de fala que roda no seu navegador e é baixado na primeira vez que você o usa

## Uma nota para celulares

Para ler respostas em voz alta em um celular, as opções OpenAI ou OpenAI-compatible são as mais confiáveis — os navegadores móveis limitam as vozes integradas.

# Notificações

As notificações avisam você quando algo precisa da sua atenção, para que você não precise ficar olhando a tela — uma sessão terminou, encontrou um erro, fez uma pergunta ou precisa de permissão para fazer algo. Configure-as em **Settings → OpenChamber → Notifications**.

## Ativar

1. Abra **Settings → OpenChamber → Notifications**.
2. Permita as notificações quando seu navegador ou sistema solicitar.
3. Escolha sobre o que você quer ser avisado:
   - uma sessão **termina**
   - uma sessão encontra um **erro**
   - uma sessão **faz uma pergunta**
   - uma sessão precisa de **permissão**
   - **subtarefas** são concluídas

## Como elas chegam até você

- no **desktop**, você recebe notificações nativas do sistema
- em um **navegador ou app instalado**, você recebe notificações web push, então elas chegam mesmo quando a aba está em segundo plano

Sessões configuradas para auto-aceitar não vão incomodá-lo com notificações de permissão.

## Personalizar o texto

Cada tipo de notificação tem um modelo de título e mensagem que você pode editar, usando campos como o nome do agente e o modelo. Há também um limite de quanto da última mensagem é incluído, para que as notificações fiquem curtas.

# Ícones de Projeto

Um ícone de projeto facilita distinguir seus projetos num relance. O OpenChamber tenta encontrar um para você, e você sempre pode definir o seu. Gerencie-o em **Settings → Projects**.

## Descoberta automática

Quando você adiciona um projeto, o OpenChamber procura por um arquivo `favicon` dentro dele e o usa como ícone do projeto. Se o seu repositório já traz um favicon, o ícone geralmente simplesmente aparece — nada a fazer.

## Definir o seu

Abra **Settings → Projects** e faça upload de uma imagem (PNG, JPEG ou SVG, até 5 MB). Uma imagem personalizada tem prioridade sobre a descoberta automática. Você também pode escolher uma cor em vez disso, ou remover a imagem para voltar à descoberta.

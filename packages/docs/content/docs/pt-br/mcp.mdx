# Servidores MCP

Um servidor MCP dá aos seus agentes ferramentas extras — coisas como pesquisar em um banco de dados, chamar uma API ou ler um serviço que você usa. Adicione-os em **Settings → MCP**.

## Adicionar um servidor

1. Abra **Settings → MCP**.
2. Adicione um servidor e escolha seu tipo:
   - **local** — o OpenChamber executa um comando na sua máquina. Você fornece o comando a ser executado e, se necessário, variáveis de ambiente.
   - **remote** — o OpenChamber conecta a uma URL hospedada por outra pessoa. Você fornece a URL e quaisquer cabeçalhos necessários (por exemplo, um token de autenticação).
3. Salve. O servidor fica ativado por padrão; você pode desativá-lo sem excluí-lo.

## Onde se aplica

Escolha o escopo ao adicionar um servidor:

- **personal** — disponível em todos os projetos
- **project** — disponível apenas no projeto atual e salvo junto com as outras configurações do projeto

Os nomes de servidor usam letras minúsculas, números, hifens e sublinhados.

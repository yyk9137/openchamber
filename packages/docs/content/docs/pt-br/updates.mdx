# Atualizações

Como você atualiza o OpenChamber depende de como o instalou. Em todos os casos, você pode verificar sua versão atual em **Settings → OpenChamber → About**.

## App de desktop

O app de desktop verifica atualizações nas releases do GitHub. Quando há uma disponível, o OpenChamber avisa você, baixa-a quando você escolhe e a instala no próximo reinício. Você está sempre no controle — nada é instalado sem a sua autorização.

## Web / CLI

Se você instalou a CLI, atualize-a a partir de **About** com os botões de verificar e atualizar, ou pelo terminal:

```bash
openchamber update
```

O OpenChamber detecta como foi instalado (npm, pnpm, yarn ou bun) e executa a atualização certa para você.

## O servidor OpenCode

O OpenChamber e o OpenCode são atualizados separadamente. Quando uma nova versão do OpenCode está disponível, o OpenChamber oferece atualizá-la e reinicia o servidor em seguida. Veja [OpenCode Server](/pt-br/opencode-server/) para como esse servidor é gerenciado.

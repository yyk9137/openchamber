# Notas, Tarefas e Planos do Projeto

Cada projeto tem seu próprio espaço de rascunho para notas, uma lista de tarefas e planos salvos. Eles pertencem ao projeto, não a uma única sessão, então permanecem conforme você se move entre sessões. Encontre-os na aba **Context** da barra lateral direita (uma aba dedicada no celular).

## Notas

Uma caixa de notas em formato livre para qualquer coisa que você queira lembrar sobre o projeto. Ela salva sozinha conforme você digita.

## Tarefas

Uma lista de verificação simples. Adicione itens, marque-os como concluídos, reordene-os arrastando e limpe os finalizados.

Cada tarefa tem um menu **send** para você entregá-la ao agente:

- envie-a para a sessão atual
- inicie uma nova sessão com ela
- inicie uma nova [sessão de worktree](/pt-br/worktrees/) com ela (apenas quando o projeto é um repositório git)

## Planos

Um lugar para manter planos mais longos como arquivos salvos. Você pode:

- importar um plano de um arquivo Markdown ou de texto
- abrir um plano para lê-lo no painel lateral
- excluir planos que você não precisa mais

Você deve voltar à aba Context com sua nota salva, sua tarefa marcada ou seu plano listado — é assim que você sabe que deu certo.

## Relacionado

- [Sessões de Worktree](/pt-br/worktrees/) — execute uma tarefa em seu próprio branch
- [Projetos](/pt-br/projects/) — estes pertencem ao projeto ativo

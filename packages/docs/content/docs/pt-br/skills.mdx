# Skills

Uma skill é um conjunto reutilizável de instruções que um agente pode incorporar quando for relevante — por exemplo, "como escrevemos mensagens de commit" ou "nossas convenções de API". Você as gerencia em **Settings → Skills**.

Para instalar skills prontas em vez de escrever as suas, veja o [Catálogo de Skills](/pt-br/skills-catalog/).

## Criar uma skill

1. Abra **Settings → Skills**.
2. Crie uma skill e dê a ela um nome e uma descrição curta. A descrição é como o agente decide quando a skill se aplica, então seja específico.
3. Escreva as instruções. Adicione arquivos de apoio se a skill precisar deles.
4. Escolha onde ela fica:
   - **personal** — disponível em todos os projetos
   - **project** — disponível apenas no projeto atual

## Usar uma skill no chat

Digite `/` no meio de uma mensagem para abrir o seletor de skills e escolha uma. O agente carrega as instruções daquela skill para a resposta.

Um `/` logo no início da mensagem abre os [comandos](/pt-br/commands-snippets/) em vez disso.

# Magic Prompts

O OpenChamber usa prompts integrados nos bastidores sempre que faz algo automaticamente — escrever uma mensagem de commit, redigir um PR, revisar uma issue, resolver um conflito, resumir uma sessão. Os Magic Prompts são onde você lê e reescreve esses prompts. Abra-os em **Settings → Magic Prompts**.

Você não precisa desta página para o uso normal. Recorra a ela quando quiser que um fluxo se comporte de forma diferente — digamos, mensagens de commit em um estilo específico.

## Editar um prompt

1. Abra **Settings → Magic Prompts**.
2. Escolha um prompt entre os grupos na barra lateral — Git, GitHub, Planning e Session.
3. Edite o texto e salve.

Alguns prompts têm uma parte visível (a mensagem que você veria) e uma parte de instruções (orientação oculta para o agente). Os prompts podem incluir `{{placeholders}}` que o OpenChamber preenche, como o diff ou o título da issue — mantenha-os no lugar.

## Redefinir

Mudou de ideia? Cada prompt tem **reset to default**, e há um **reset all** se você quiser recomeçar em tudo.

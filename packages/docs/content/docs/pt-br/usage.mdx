# Uso e Cotas

A página de Uso mostra quanto de cada plano de provedor você já usou, para que você veja o quão perto está de um limite. Abra-a em **Settings → Usage**.

## O que você vê

Para cada provedor conectado, o OpenChamber mostra:

- quanto da janela atual você usou, como uma barra
- um detalhamento por modelo
- um indicador de ritmo, para você saber se está a caminho de esgotar

Você pode escolher quais provedores aparecem, e o mesmo resumo está disponível em um menu suspenso no cabeçalho do app.

## Provedores suportados

O uso funciona para os provedores que publicam uma cota, incluindo Claude, Codex, GitHub Copilot, Google, OpenRouter, Kimi, NanoGPT, z.ai, Zhipu, MiniMax, Ollama Cloud e Wafer.

Um provedor só mostra o uso depois que você fez login nele na página de [Provedores](/pt-br/providers/). Alguns provedores precisam de um passo extra — por exemplo, o Ollama Cloud lê um arquivo de sessão que você configura separadamente. Se um provedor não mostra dados, geralmente significa que essa credencial extra está faltando.

# 项目操作

项目操作是一个你保存一次、点击即可运行的 shell 命令 — 你的开发服务器、一次构建、一次测试运行。每个项目都保留自己的列表。在 **Settings → Projects → Project Actions** 中设置它们。

## 添加操作

1. 打开 **Settings → Projects** 并找到 **Project Actions** 部分。
2. 添加一个操作，给它一个名称，选择一个图标，并输入要运行的命令。
3. 保存。

如果某个命令只在特定操作系统上才有意义，你可以将该操作限制在某些操作系统上。

## 运行操作

操作位于应用标题栏的一个菜单中。点击其中一个，OpenChamber 就会在你项目文件夹的终端中运行它，并将你切换到终端视图，让你可以观察输出。从同一菜单可以停止它。

## 自动打开开发服务器

为一个启动服务器的操作开启 **auto-open URL**。OpenChamber 会监视输出中的本地地址并提议打开它 — 参阅 [预览与开发服务器](/zh-cn/preview/)。在桌面端，你还可以通过 SSH 端口转发来路由它。

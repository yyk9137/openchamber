# 项目笔记、待办与计划

每个项目都有自己的草稿空间，用于存放笔记、待办列表和已保存的计划。它们属于项目，而不属于任何单个会话，因此当你在会话之间切换时它们会一直保留。在右侧边栏的 **Context** 标签页中可以找到它们（在移动端是一个专用标签页）。

## 笔记

一个自由格式的笔记框，用于记录任何你想记住的关于项目的内容。它会在你输入时自动保存。

## 待办

一个简单的清单。添加条目、勾选完成、通过拖动重新排序，并清除已完成的条目。

每个待办都有一个 **send** 菜单，让你可以把它交给智能体：

- 发送到当前会话
- 用它开启一个新会话
- 用它开启一个新的 [worktree 会话](/zh-cn/worktrees/)（仅当项目是 git 仓库时）

## 计划

一个用于将较长的计划保存为文件的地方。你可以：

- 从 Markdown 或文本文件导入计划
- 打开一个计划在侧面板中阅读
- 删除你不再需要的计划

完成后你应该会回到 Context 标签页，并看到笔记已保存、待办已勾选或计划已列出 — 这就是它生效的标志。

## 相关内容

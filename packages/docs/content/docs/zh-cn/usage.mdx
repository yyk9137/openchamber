# 用量与配额

用量页面显示你已使用的每个提供商套餐的额度，让你能看到自己距离限额还有多远。在 **Settings → Usage** 中打开它。

## 你能看到什么

对于每个已连接的提供商，OpenChamber 会显示：

- 当前周期内你已使用的量，以进度条形式展示
- 按模型的细分
- 一个速率指示器，让你判断自己是否会用尽额度

你可以选择哪些提供商出现在此处，相同的摘要也可以从应用标题栏的下拉菜单中查看。

## 支持的提供商

用量功能适用于那些公布配额的提供商，包括 Claude、Codex、GitHub Copilot、Google、OpenRouter、Kimi、NanoGPT、z.ai、Zhipu、MiniMax、Ollama Cloud 和 Wafer。

只有在你于 [提供商](/zh-cn/providers/) 页面登录某个提供商后，它才会显示用量。少数提供商需要额外的步骤 — 例如，Ollama Cloud 会读取一个你单独设置的会话文件。如果某个提供商没有显示数据，通常意味着缺少这个额外的凭证。

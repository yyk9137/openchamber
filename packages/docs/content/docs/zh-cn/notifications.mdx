# 通知

通知会在有事情需要你关注时告诉你，这样你就不必盯着屏幕 — 会话完成了、遇到错误、提出了问题，或需要权限来做某件事。在 **Settings → OpenChamber → Notifications** 中设置它们。

## 开启它们

1. 打开 **Settings → OpenChamber → Notifications**。
2. 当你的浏览器或系统询问时，允许通知。
3. 选择你想被告知哪些事项：
   - 会话**完成**
   - 会话遇到**错误**
   - 会话**提出问题**
   - 会话需要**权限**
   - **子任务**完成

## 它们如何送达你

- 在**桌面端**，你会收到原生系统通知
- 在**浏览器或已安装的应用**中，你会收到网页推送通知，因此即使标签页在后台它们也能送达

设置为自动接受的会话不会用权限通知来打扰你。

## 自定义措辞

每种通知都有一个标题和消息模板，你可以编辑它们，使用诸如智能体名称和模型之类的字段。还有一个对最后一条消息包含多少内容的限制，以使通知保持简短。

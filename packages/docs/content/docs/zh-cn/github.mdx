# GitHub Issue 与 PR

连接你的 GitHub 账号，OpenChamber 就可以拉取 issue 和拉取请求、直接从其中一个开始会话，并为你打开或更新 PR。

## 连接 GitHub

1. 打开 **Settings → Git**。
2. 在 GitHub 下，选择 **Connect**。OpenChamber 会显示一个链接和一个短码。
3. 打开链接，输入短码并批准。

连接后，你的账号会显示在 GitHub 部分下。你可以连接多个账号并在它们之间切换，或随时断开连接。

## 从 issue 或 PR 开始工作

当你在连接了 GitHub 的情况下创建 [worktree 会话](/zh-cn/worktrees/) 时，你可以选择 **Start from GitHub issue/PR**：

- 选择一个 **issue**，OpenChamber 会以它命名分支，并以该 issue 及其评论作为第一条消息打开会话
- 选择一个 **pull request**，它会检出该 PR 的分支；你可以包含 PR 的 diff，以便智能体掌握完整的变更

这会让你直接进入一个已经加载好上下文的会话。

## 打开并管理拉取请求

在 [git 视图](/zh-cn/git/) 的 **PR** 标签页中，你可以创建拉取请求、更新它、将草稿标记为就绪或合并它。OpenChamber 可以从你的变更生成 PR 标题和描述。

# Modo de voz

El modo de voz te permite dictar mensajes y que las respuestas se te lean en voz alta. Actívalo en **Settings → OpenChamber → Voice**.

## Leer respuestas en voz alta (texto a voz)

Elige cómo se pronuncian las respuestas:

- **browser** — las voces integradas de tu navegador, sin configuración
- **OpenAI** — las voces de OpenAI; pega tu clave de API y elige una voz
- **compatible con OpenAI** — cualquier servicio que hable el formato de OpenAI; introduce su URL y, si la necesita, una clave de API
- **macOS say** — el comando `say` integrado, en los Mac que lo tienen

Una vez activado, obtienes un botón de reproducción en los mensajes para escucharlos en voz alta.

## Dictar mensajes (voz a texto)

Elige cómo se transcribe tu voz:

- **browser** — el reconocimiento integrado de tu navegador, sin configuración
- **server** — un servicio de transcripción compatible con OpenAI; introduce su URL y, si hace falta, una clave de API
- **on-device** — un modelo de voz que se ejecuta en tu navegador y se descarga la primera vez que lo usas

## Una nota para teléfonos

Para leer respuestas en voz alta en un teléfono, las opciones de OpenAI o compatible con OpenAI son las más fiables: los navegadores móviles limitan las voces integradas.

# Servidores MCP

Un servidor MCP da a tus agentes herramientas adicionales: cosas como buscar en una base de datos, llamar a una API o leer un servicio que usas. Añádelos en **Settings → MCP**.

## Añade un servidor

1. Abre **Settings → MCP**.
2. Añade un servidor y elige su tipo:
   - **local** — OpenChamber ejecuta un comando en tu máquina. Le das el comando que debe ejecutar y, si hace falta, variables de entorno.
   - **remote** — OpenChamber se conecta a una URL que aloja otra persona. Le das la URL y cualquier encabezado que necesite (por ejemplo, un token de autenticación).
3. Guarda. El servidor está activo de forma predeterminada; puedes desactivarlo sin eliminarlo.

## Dónde se aplica

Elige el ámbito al añadir un servidor:

- **personal** — disponible en todos los proyectos
- **proyecto** — disponible solo en el proyecto actual, y guardado junto con los demás ajustes del proyecto

Los nombres de los servidores usan letras minúsculas, números, guiones y guiones bajos.

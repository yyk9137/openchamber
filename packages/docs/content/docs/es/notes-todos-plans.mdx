# Notas, tareas y planes de proyecto

Cada proyecto tiene su propio espacio de trabajo para notas, una lista de tareas y planes guardados. Pertenecen al proyecto, no a ninguna sesión concreta, así que permanecen mientras te mueves entre sesiones. Los encuentras en la pestaña **Context** de la barra lateral derecha (una pestaña dedicada en el móvil).

## Notas

Un cuadro de notas de formato libre para cualquier cosa que quieras recordar sobre el proyecto. Se guarda solo a medida que escribes.

## Tareas

Una lista de verificación sencilla. Añade elementos, márcalos como completados, reordénalos arrastrando y elimina los terminados.

Cada tarea tiene un menú **send** para que se la entregues al agente:

- enviarla a la sesión actual
- iniciar una sesión nueva con ella
- iniciar una nueva [sesión de worktree](/es/worktrees/) con ella (solo cuando el proyecto es un repositorio de git)

## Planes

Un lugar para guardar planes más largos como archivos. Puedes:

- importar un plan desde un archivo Markdown o de texto
- abrir un plan para leerlo en el panel lateral
- eliminar los planes que ya no necesites

Deberías volver a la pestaña Context con tu nota guardada, tu tarea marcada o tu plan listado: así sabes que funcionó.

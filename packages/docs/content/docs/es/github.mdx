# Issues y PRs de GitHub

Conecta tu cuenta de GitHub y OpenChamber podrá traer issues y pull requests, iniciar una sesión directamente desde uno, y abrir o actualizar PRs por ti.

## Conecta GitHub

1. Abre **Settings → Git**.
2. En GitHub, elige **Connect**. OpenChamber muestra un enlace y un código corto.
3. Abre el enlace, introduce el código y aprueba.

Cuando esté conectado, tu cuenta aparece en la sección de GitHub. Puedes conectar más de una cuenta y cambiar entre ellas, o desconectar en cualquier momento.

## Empieza a trabajar desde una issue o PR

Cuando creas una [sesión de worktree](/es/worktrees/) con GitHub conectado, puedes elegir **Start from GitHub issue/PR**:

- elige una **issue** y OpenChamber nombra la rama según ella y abre la sesión con la issue y sus comentarios como primer mensaje
- elige un **pull request** y hace checkout de la rama del PR; puedes incluir el diff del PR para que el agente tenga el cambio completo

Esto te deja directamente en una sesión con el contexto ya cargado.

## Abre y gestiona pull requests

Desde la pestaña **PR** en la [vista de git](/es/git/), puedes crear un pull request, actualizarlo, marcar un borrador como listo o fusionarlo. OpenChamber puede generar el título y la descripción del PR a partir de tus cambios.

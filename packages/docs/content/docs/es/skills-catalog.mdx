# Catálogo de skills

El catálogo de skills te permite instalar skills que otras personas han publicado en lugar de escribir las tuyas. Ábrelo en **Settings → Skills → Catalog**.

Para escribir tus propias skills, consulta [Skills](/es/skills/).

## Instala una skill

1. Abre el catálogo.
2. Explora las fuentes integradas —el repositorio de skills de Anthropic y el registro comunitario de ClawdHub— o busca.
3. Elige una skill e instálala.
4. Elige dónde instalarla: para todo lo que hagas, o solo en el proyecto actual.

Si ya existe una skill con el mismo nombre, OpenChamber pregunta qué hacer: omitirla, sobrescribirla o decidir skill por skill.

## Añade tu propia fuente

Puedes añadir cualquier repositorio de Git como fuente usando su nombre `owner/repo` o una URL de Git completa. Los repositorios privados necesitan tener el acceso configurado en tu máquina (una clave SSH o credenciales guardadas); si una fuente no puede autenticarse, el catálogo lo indica en lugar de fallar en silencio.

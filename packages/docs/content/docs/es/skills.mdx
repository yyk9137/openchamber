# Skills

Una skill es un conjunto reutilizable de instrucciones que un agente puede incorporar cuando es relevante; por ejemplo, "cómo escribimos los mensajes de commit" o "nuestras convenciones de API". Las gestionas en **Settings → Skills**.

Para instalar skills ya hechas en lugar de escribir las tuyas, consulta [Catálogo de skills](/es/skills-catalog/).

## Crea una skill

1. Abre **Settings → Skills**.
2. Crea una skill y dale un nombre y una descripción breve. La descripción es lo que el agente usa para decidir cuándo aplica la skill, así que hazla específica.
3. Escribe las instrucciones. Añade archivos de apoyo si la skill los necesita.
4. Elige dónde vive:
   - **personal** — disponible en todos los proyectos
   - **proyecto** — disponible solo en el proyecto actual

## Usa una skill en el chat

Escribe `/` en medio de un mensaje para abrir el selector de skills y elige una. El agente carga las instrucciones de esa skill para la respuesta.

Una `/` al principio del mensaje abre los [comandos](/es/commands-snippets/) en su lugar.

# Notificaciones

Las notificaciones te avisan cuando algo necesita tu atención para que no tengas que vigilar la pantalla: una sesión terminó, encontró un error, hizo una pregunta o necesita permiso para hacer algo. Configúralas en **Settings → OpenChamber → Notifications**.

## Actívalas

1. Abre **Settings → OpenChamber → Notifications**.
2. Permite las notificaciones cuando tu navegador o sistema lo pida.
3. Elige sobre qué quieres recibir avisos:
   - una sesión **termina**
   - una sesión encuentra un **error**
   - una sesión **hace una pregunta**
   - una sesión necesita **permiso**
   - se completan **subtareas**

## Cómo te llegan

- en el **escritorio**, recibes notificaciones nativas del sistema
- en un **navegador o app instalada**, recibes notificaciones web push, así que llegan incluso cuando la pestaña está en segundo plano

Las sesiones configuradas para aceptar automáticamente no te molestarán con notificaciones de permiso.

## Personaliza el texto

Cada tipo de notificación tiene una plantilla de título y mensaje que puedes editar, usando campos como el nombre del agente y el modelo. También hay un límite de cuánto del último mensaje se incluye, para que las notificaciones queden cortas.

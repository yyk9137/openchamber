# Uso y cuotas

La página de uso muestra cuánto del plan de cada proveedor has consumido, para que veas qué tan cerca estás de un límite. Ábrela en **Settings → Usage**.

## Lo que ves

Para cada proveedor conectado, OpenChamber muestra:

- cuánto de la ventana actual has usado, como una barra
- un desglose por modelo
- un indicador de ritmo, para saber si vas camino de agotarlo

Puedes elegir qué proveedores aparecen, y el mismo resumen está disponible desde un menú desplegable en el encabezado de la app.

## Proveedores compatibles

El uso funciona con los proveedores que publican una cuota, entre ellos Claude, Codex, GitHub Copilot, Google, OpenRouter, Kimi, NanoGPT, z.ai, Zhipu, MiniMax, Ollama Cloud y Wafer.

Un proveedor solo muestra el uso una vez que has iniciado sesión en él en la página de [Proveedores](/es/providers/). Algunos proveedores necesitan un paso extra: por ejemplo, Ollama Cloud lee un archivo de sesión que configuras por separado. Si un proveedor no muestra datos, normalmente significa que falta esa credencial adicional.
